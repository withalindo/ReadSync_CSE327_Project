\doxysubsection{Detailed Description}
Frontend page for selecting books to borrow. 

This page provides a form for users to select books for borrowing. It fetches the list of available books from the database and displays them in a table with checkboxes for selection. The form submits the selected books and the user\textquotesingle{}s username to the backend script {\ttfamily \doxylink{borrow_book_8php}{borrow\+Book.\+php}}.


\begin{DoxyItemize}
\item Connects to the database using {\ttfamily connect\+DB.\+php}.
\item Queries the {\ttfamily books} table for available books, retrieving {\ttfamily book\+\_\+id}, {\ttfamily book\+\_\+name}, and {\ttfamily book\+\_\+price}.
\end{DoxyItemize}
