This file contains the implementation of the updateBookCopy class, which I created to manage book copy updates in the database.

I designed this file to handle operations like checking if a book exists by its ISBN and updating the number of copies for a book. It also processes POST requests to update book copies and redirects users to the appropriate page after the operation.
