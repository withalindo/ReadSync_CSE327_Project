\doxysubsection{Detailed Description}
This file establishes a connection to the database. 

I created this file to handle the database connection for the application. It connects to the {\ttfamily library\+Management\+System\+DB} database using the provided credentials.


\begin{DoxyItemize}
\item I defined the database server, username, password, and database name as variables.
\item I used the {\ttfamily mysqli\+\_\+connect} function to establish the connection.
\item If the connection fails, I catch the exception and display an error message.
\item If the connection is successful, I display a success message (for debugging purposes).
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item This file is included in other backend scripts to provide a shared database connection.
\item Make sure to replace the placeholder password ({\ttfamily Password}) with the actual database password.
\end{DoxyItemize}
\end{DoxyNote}
